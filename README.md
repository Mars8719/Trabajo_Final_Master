
Este repositorio contiene el desarrollo del Trabajo de Fin de Máster (TFM), en el cual se aplican los conocimientos adquiridos durante el programa académico. El proyecto incluye el análisis, diseño e implementación de una solución orientada a resolver una problemática específica, utilizando herramientas y metodologías actuales.

El objetivo principal es integrar conceptos teóricos y prácticos, evidenciando competencias en el área de estudio y aportando una propuesta funcional.
